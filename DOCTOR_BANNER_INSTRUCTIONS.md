# Doctor Banner Image Setup

## Where to Save the Banner Image

1. **Navigate to the drawable folder:**
   ```
   app/src/main/res/drawable/
   ```

2. **Save your banner/doctor illustration image:**
   - **Filename:** `doctor_banner.png` (or `doctor_banner.jpg`)
   - **Recommended size:** 1080x600 pixels or similar landscape ratio
   - **Format:** PNG (with transparency) or JPG
   - **Location:** `C:\Users\Sujal\Desktop\LivHealth\app\src\main\res\drawable\doctor_banner.png`

## After Saving

1. **Sync Gradle** in Android Studio:
   - File → Sync Project with Gradle Files

2. **Rebuild** the project:
   - Build → Rebuild Project

3. The banner will appear above the "Scan QR" button in the Doctor Dashboard.

## Image Suggestions

- A doctor/veterinarian illustration
- Medical-themed banner with animal care elements
- Professional healthcare banner design
- Vet clinic branding image

The banner will be displayed with rounded corners and shadow effects to match the app's modern design.
